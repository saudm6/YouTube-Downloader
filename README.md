# YouTube-Downloader
Download YouTube videos using python.

# Windows

1) Install pytube on your PC/Mac by using the command <br>
    ```pip install pytube``` for Windows & ```pip3 install pytube for Mac``` 
3) Change the line 14 from "D:/youtube/videos" to the file you would like the video to download.
4) Finally you are going to open the terminal on you IDE and type ```py Youtube-Downloader.py https://www.youtube.com/watch?v=-qJiDPljeB4``` (Windows).
   ```python3 Youtube-Downloader.py https://www.youtube.com/watch?v=-qJiDPljeB4``` (Mac). Replace the youtube link with whatever video you would like to download. Finally if you decide on changing the name of the python file then you should change the final command to ```py (File-Name) (video-url)``` for Windows and ```python3 (File-Name) (video-url)``` for Mac.
