# YouTube-Downloader
Download YouTube videos using python.

# Instructions

1) First you will need to install pytube on your PC/Mac by using the command __*"pip install pytube"*__ for Windows & __*"pip3 install pytube for Mac"*__. 
2) Then you are going to change the line 14 from "D:/youtube/videos" to the file you would like the video to download.
3) Finally you are going to open the terminal on you IDE and type ```python Youtube-Downloader.py https://www.youtube.com/watch?v=-qJiDPljeB4``` (Windows).
   __*"python3 Youtube-Downloader.py https://www.youtube.com/watch?v=-qJiDPljeB4"*__ (Mac). Replace the youtube link with whatever video you would like to download. Finally if you decide on changing the name of the python file then you should change the final command to "python (File-Name) (video-url)" for Windows and "python3 (File-Name) (video-url)" for Mac.

Enjoy !!!
